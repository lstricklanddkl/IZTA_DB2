Loading Files

To load files:

  1. Click on "Load Files" in menu bar, to open the file browser
  2. Select the files you want to open
  3. Click on the "Open" button to close the dialog

Once files have been loaded, it will take a little while for the plot to be generated. If no plot is generated, either more data needs to be loaded, or there is an error in the data

List of Tables

A list of all tables is provided. The top 10 tables will be shown on the plot by default.

Change label:to change the label of a table, click on the icon, edit the label and select "Update Label" button. Note this modal shows other information about the table

Change color:to change the color of an object, click on the current color and select a color from the list of options

Hide table:to hide a table, click on the icon, and that path will be hidden

Show table:to show a hiddentable, click on the icon, and that path will be shown

Change Time

To change the start and end times to be plotted, drag the handles of the sliders below the graph

Take Screenshot

Clicking on the "Screenshot" in the menu bar will download a png image of the Plot

Generate Report

Clicking on the "Report" in the menu bar will generate a report for each of the tables selected

Download

Clicking on the "Download" in the menu bar will download a PDF of the report generated in the step above. Note adjusting the size of the browser window, will change the layout of this report, since this function is simply generating a PDF copy of what is show on the screen.
